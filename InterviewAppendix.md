# Interview Record (Mock)

- Date: 2025-09-08
- Interviewer: Module Leader
- Interviewee: Client (Bank representative)

## Key Points
- Customers can hold multiple accounts concurrently.
- Savings Account: deposits only; monthly interest 0.05%.
- Investment Account: deposits and withdrawals; monthly interest 5%; requires initial BWP500.00 to open.
- Cheque Account: deposits and withdrawals; only for working customers; requires employer name and address.
- Interest applied monthly on interest-bearing accounts.
- System should expose registration, account opening, deposit/withdraw, and interest application.
- Security and correctness prioritized; no GUI/database in first iteration.
